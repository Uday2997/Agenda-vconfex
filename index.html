Going With AGENDA

The agenda plays the most important part in the event as it allows users to have a quick
glimpse on the list of the sessions scheduled in the event.The agenda for the event can be
created and scheduled from the backend.

There are following subfeatures in the Agenda Section–

Add Halls

Edit Hall Details:

  • Hall Creative -> a Hall Video Placement -> Moderate Comment -> Add Moderators

Add New Session:

  • Bulk upload
  • Add session
    • Title of the session -> Session start date & time -> Session end date & time -> Video source (VOD Live, Youtube Live, Vimeo Live & VConfex Live) -> Upload video/Select video from the media center -> Select the speaker of the sessions from the dropdown menu -> Public Chat on/off for the session -> Moderate Comment on/off for the session -> Description of the session

1. Add Halls

In the Add hall tab you can manage the number of halls and their sessions according to the event, After clicking on add hall tab on the left side you will redirect to the below screen.
On this screen you can see there is one hall named Auditorium. Under Auditorium you can add sessions for your event.

2. Edit Hall Details

There are following steps involved in updating hall details-

Please click on the edit hall details.

In the Edit hall details, you can edit the hall name, set the hall video placement type, select hall creatives from library (system folder) or select default screens from template gallery and on or off the moderate comment for the auditorium.

After entering all the fields then click on Save & Continue button to add the moderators for the Auditorium. Otherwise click on Save button to save the Hall and the popup will close by itself

If in case the auditorium creates extra or you have to remove from the agenda section then simply click on delete icon.

Or if in case the add hall button clicks then just click on the cancel button to close the popup.

Now the next step is to add the Moderators for the Auditorium. After clicking on Save and Continue button, then the popup will open to add the moderators.

Here are the steps to be followed:

  • Add first name -> Add last name -> Choose the profile image for the moderator in the given dimensions of (120px to 40px) -> Enter the email address -> Add mobile number -> Add company name -> Add designation of the moderator

Note: Some fields are marked as asterisk(*)/mandatory which means that fields are mandatory to enter.

After filling all the fields, just click on the add moderators button to save. (refer screen 8 above)

3. Add New Session

There are two types to add the sessions in the Agenda of the auditorium:

Adding the sessions manually

Bulk upload

3.a Adding sessions manually

Click on Add Session, then again click on add session button to open the popup.

Now follow below steps for adding the session to the auditorium:

  • Add title of the session
  • Select the start date & time of the session (you can enter the date & time manually or
    select from the calendar when you click on the tab).
  • Select the end date & time of the session (you can enter the date & time manually or
    select from the calendar when you click on the tab).
  • Select the video source (VOD Live, Youtube Live, Vimeo Live & VConfex Live)
  • Select the speakers from the dropdown menu (which are added in the users section - This will be discussed later)
  • Select public chat On/Off
  • Select moderate comment Yes/No
  • Add description which is optional
  • After entering all the fields, click on the save button to save the session.

3b. Bulk Upload

There are three steps to bulk upload the sessions in the Agenda of the auditorium:

Click on Add Session, then click on Bulk upload button to open the popup.

A pop-up will appear on the right side of the screen, there you will find a download sample CSV option, by clicking on that a sample file will be downloaded in your system which contains the supportive format of bulk agenda upload, fill up the agenda in CSV according to sample file then click on upload button.

And then save the agenda, your agenda will be automatically uploaded to the respective hall.

4. View Add Moderators

There are five steps to bulk upload the sessions in the Agenda of the auditorium:

First you have to click on View/Add Moderators. Then the other screen will open to add the moderators.

Click on Add Moderators button the popup will open from the right side.

After filling in all the required details your moderator will be added to the particular hall.

At last click on the save button to add the moderator. (refer to screen 18)

You can also edit or delete the moderators.

Add Other Halls

If you want to add more than one hall, click on the Add Hall button on the right side of your screen, by clicking on that a pop-up will appear (refer to screen 5), just fill up all the options according to your preference and another hall will be added to your event.

Video uploading in agenda:

For video uploading in the agenda, first of all, you need to add the session (refer to screen 13[a]). There are following steps and methods to upload the different types of videos in agenda.

Vconfex Player(If the video is recorded):

Select the type of video Vconfex Player by default it will already be selected. By clicking on upload video, a pop-up will appear to select the video from within your local system’s files, just select the session-specific video and click on upload, after successfully uploading the video click on the save button. Now your video will be added to the session

Youtube(If the video is on youtube):

Select the video type youtube, by selecting youtube a video URL box will appear, paste the video embedded URL there and click on the Save button. Your video will be added to the session.

Youtube Live(If the video is Live on Youtube):

Select the video type Youtube Live, by selecting this option a video URL box will appear, paste the embedded youtube live URL and click on the Save button, Your live video will be added to the session.

*Note–For extracting the embedded youtube live URL please open any live video on youtube and click on share button below the video and click on embed and then embed code will be displayed from which you need to copy img src and paste it in url box at backend*.

Vimeo Player(If the video is on Vimeo Player):

Select the video type Vimeo Player, by selecting this option a video URL box will appear, paste the Vimeo URL of the video there, and click on the Save button, Your video will be added to the session.

VConfex Live(If the video is on VConfex Live):

Select the video type VConfex Live, by selecting this option a video URL box will appear, paste the live stream key provided by the Vconfex team there and click on the Save button, the live session will be added to the session.

For Making live the uploaded video:

This feature is only available with uploaded video. If you want to make the uploaded video live, First of all, add the session by uploading the video, save the session. Now click on (four dots) on the right corner of the session name and click on the make live option, after submitting the declaration your video will be scheduled as a live video on the platform.

Manage Sessions

In the manage session tab, you can add the session, edit the session, add the quiz, view the session and delete the session.

Adding quiz in session:

For adding a quiz in any session, click on the (three line button) on the right corner of the session name and select add quiz option there. After clicking on the add quiz it will redirect you to the Quiz section. There you can add quizzes by entering the quiz name and quiz description. After adding the quiz it will redirect you to the add question section. There you can add the questions and answer options. After successfully adding the quiz within the maximum question limit of the quiz. The quiz will be added to the session.

If you want to add more questions then click on Add New Question. (Refer to screen 22)

Here you can also add the quiz selected from the media center.
